WebGL Scene same as document.body has to be in absolute non-dependent position. Also [ z-index: value ] always for ancestors only, which conditionally set [ : value ] whether to -1 or 1, if -1 when the scene becomes as background and and the rest content becomes as foreground automatically!
Descendant content such as buttons that overlap the WebGL Scene must be in [position: relative]. It does not need z-index for itself to make it as foreground. This job is done by ancestors z-index only!
